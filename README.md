# CapTable Generator

## Overview

The CapTable Generator is a web application designed to help startups and companies manage their capitalization tables (cap tables). This tool allows users to input their company's equity data, visualize the distribution of shares, and generate reports for stakeholders.

## Features

- **Equity Management**: Track and manage different types of equity, including common stock, preferred stock, and options.
- **Visualization**: Generate visual representations of the cap table, including pie charts and bar graphs.
- **Reporting**: Create detailed reports for investors, board members, and other stakeholders.
- **User-Friendly Interface**: Intuitive design for easy navigation and data input.
- **Data Export**: Export cap table data to various formats, including CSV and Excel.

## Technologies Used 

- **Frontend**: React, Tailwind CSS, Radix UI
- **Backend**: Node.js, Express
- **Database**: MongoDB `(DB need to maintain)`
- **State Management**: React Query
- **Charting**: Recharts
- **Routing**: React Router
- **Form Handling**: React Hook Form
- **Date Handling**: Date-fns
- **Excel Handling**: xlsx

## Installation

To get started with the CapTable Generator, follow these steps:

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/blue-lotus-org/captable-generator.git
   cd captable-generator
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Run the Development Server**:
   ```bash
   npm run dev
   ```

4. **Build for Production**:
   ```bash
   npm run build
   ```

5. **Preview the Build**:
   ```bash
   npm run preview
   ```

## Usage

1. **Access the Application**: Open your web browser and navigate to `http://localhost:3000`.
2. **Input Data**: Use the intuitive interface to input your company's equity data.
3. **Generate Reports**: Use the reporting tools to generate detailed reports for stakeholders.
4. **Export Data**: Export your cap table data to CSV or Excel for further analysis. (CSV need to maintain)
